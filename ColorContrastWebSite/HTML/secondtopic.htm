Click the Extension button near the address bar of Google browser.
This popup appears:



Click the Pick Color button. The popup disapears, then the picker and the picker toolbar apears on the tested page.
Move the pickup cursor (cross cursor) over the page until you aim for the color you want to inspect. Look in the center of the magnifyglass or in the color-preview square of the toolbar for the desired color.
Left-click to record the color as a background color or Right-click - for the foreground.

The last picked color is availble in the clipboard for easy pasting in your project (but this behavior may be changed in the Options Page.)
When you've done your selections for foreground and background colors,
click again the extension button near the address bar. The extension popup opens once more with the your selections in the right edit boxes. The color format is '#rrggbb' where 'rr', 'gg' and 'bb' are two hexadecimal digit for red, green and blue levels (0 to 255 decimal.)
Under each edit box the extension displays the decoded name for the color either as an exact value or 'close to.' Some codes have multiple names (as in 'Cyan' and 'Aqua'.) The extension will display both.

Decode bars under each edit box have buttons to pin the value for editing. When the decoded value is 'close to' a color name, pinning will replace the actual color with the named one.

When needed and possible, the extension tries to resolve six (or less) sets of colors compliant with AAA, AA and A standard. Note that this is not always possible, so you may end up with 4, 3, 2 ... sets instead of 6. (For each standard, we try to rezolve a set changing the foreground and another changing the background.)

Each suggested color set displays the foreground and background color codes, the contrast and have an <kbd>OK</kbd> button to accept those values in the edit boxes.
